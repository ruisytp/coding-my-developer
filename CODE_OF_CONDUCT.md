# 行为准则

## 我们的承诺

我们致力于营造一个开放、友好、多样化的社区环境，无论年龄、体型、身体能力、种族、性别认同和表达、经验水平、教育背景、社会经济地位、国籍、个人外貌、种族、宗教信仰或性取向。

## 我们的标准

**积极的行为包括：**
- 使用友好和包容的语言
- 尊重不同的观点和经历
- 耐心接受建设性的批评
- 关注社区的最佳利益
- 展现对他人的同理心

**不可接受的行为包括：**
- 使用性暗示语言或图像，以及不受欢迎的性关注或挑逗
- 侮辱、贬低或人身攻击评论
- 公开或私下骚扰
- 发布他人的私人信息（如物理地址或电子地址），未经明确许可
- 其他在专业环境中被认为不合适的行为

## 我们的职责

项目维护者有责任澄清可接受行为的标准，并有权删除、编辑或拒绝不符合本行为准则的评论、提交、代码、维基编辑、问题和其他贡献。

项目维护者有权利和责任对违反本行为准则的不当、威胁、冒犯或有害行为采取适当和公平的纠正措施。

## 范围

本行为准则适用于项目空间，也适用于在公共空间中代表项目或其社区的个人。代表项目可能包括使用官方电子邮件地址、通过官方社交媒体账户发帖，或在在线或线下活动中担任指定代表。

## 执行

如果您目睹或遭受了不可接受的行为，请联系项目团队：your.email@example.com。

所有投诉都将得到及时和公平的调查和回应。

项目维护者不遵守或执行行为准则可能会面临临时或永久的后果，由项目领导层决定。
